Home
====
This is a Python implementation of the Path Check Foundation Paper-first
Verifiable Vaccination Credentials system. The spec is defined
`here <https://github.com/path-check/paper-cred>`_.

This library is currently under heavy development, and currently unusable.
It is being developed `here <https://github.com/armaanb/python-paper-cred>`_.
